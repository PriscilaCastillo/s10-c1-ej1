1. Que es una funcion y para que sirve

SUna serie de instrucciones agrupadas en un mismo proceso, esta funcion puede ser llamada despues en cualquier momento para que realice la accion asignada y no se tenga que codear lo mismo de manera repetida.

2. Que es una funcion por declaracion y expresion y en que se diferencian

Las funciones por declaracion son funciones que se elevan al maximo nivel y se pueden llamar antes de haberlas declarado y se expresan de la siguiente manera:
funcion sumar (parametro1, parametro2) {}

Las funciones por expresion son mas recomendadas de usar, solo se pueden llamar despues de haberlas declarado, se expresan de la siguiente manera:
const functionExp = (parametro1, parametro2) => { }

3. Como crear , invocar, definir parametros y argumentos

Crear la funcion:
function sumar(a, b) {
return a + b;
}

Invocar una funcion:
sumar (a; b);

4. Como realizar comunicacion entre funciones

Se definen las funciones y se puede llamar una dentro de otra y definir la manera en la que van a interactuar, por ejemplo, con un if else.
